# 01 · Deep Learning

Neural networks from first principles. Without understanding this, using PyTorch is just copy-pasting.

## Exercises

| Exercise | File | Description |
|---|---|---|
| 1 | `01_perceptron.py` | Implement a perceptron from scratch |
| 2 | `02_backprop.py` | Manual backpropagation on a small network |
| 3 | `03_pytorch_basics.py` | Tensors, autograd, basic training loop |
| 4 | `04_cnn_cifar.py` | CNN to classify CIFAR-10 images |
| 5 | `05_regularization.py` | Compare models with/without dropout and batch norm |
